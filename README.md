# E-Commerce Project

આ એક Next.js 16 (App Router) બેઝ્ડ ઈ-કોમર્સ વેબસાઈટ છે.

## Features
- Admin Panel (Products, Orders, Blogs, Settings)
- Stripe Checkout
- Shopify Shipping Integration
- Cloudinary Image Upload
- NextAuth Login

## How to Run Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.
